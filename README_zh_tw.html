# Awesome Nano-Banana 圖片庫 - 繁體中文互動版 ## 🍌 功能特色 這是一個完全互動的繁體中文版 Nano-Banana 圖片庫，讓您可以輕鬆瀏覽和試用各種 AI 圖片生成案例。 ### ✨ 主要功能 - **🔍 智能搜尋**：根據標題、作者或提示詞內容搜尋案例 - **🏷️ 分類篩選**：按人物、建築、藝術、產品、修圖等類別篩選 - **🖼️ 圖片預覽**：點擊案例卡片查看詳細的輸入/輸出圖片 - **📋 一鍵複製**：複製提示詞到剪貼板，直接貼到 Gemini 使用 - **🔗 Gemini 整合**：直接跳轉到 Gemini 網站試用提示詞 - **📱 響應式設計**：支援手機、平板、桌面等各種設備 ### 🚀 使用方法 1. **開啟檔案**：雙擊 `index_zh_tw.html` 在瀏覽器中開啟 2. **瀏覽案例**：滾動頁面查看所有案例，或使用搜尋框快速找到特定案例 3. **查看詳情**：點擊任何案例卡片查看完整的輸入/輸出圖片 4. **複製提示詞**：點擊「複製提示詞」按鈕，然後在 Gemini 中貼上使用 5. **試用案例**：點擊「在 Gemini 中試用」按鈕直接跳轉到 Gemini 網站 ### 🎯 案例分類 - **人物相關**：人物換衣、髮型更換、姿勢修改、表情控制等 - **建築相關**：3D建築生成、等距視圖、地圖轉換等 - **藝術創作**：插畫轉換、角色設定、漫畫製作等 - **產品設計**：包裝設計、信息圖、拆解圖等 - **修圖編輯**：自動修圖、照片上色、水印添加等 ### 💡 使用技巧 1. **搜尋功能**：可以搜尋案例標題、作者名稱或提示詞內容 2. **分類篩選**：使用頂部的篩選按鈕快速找到特定類型的案例 3. **圖片複製**：在詳細視圖中可以複製圖片網址 4. **提示詞優化**：複製提示詞後可以在 Gemini 中進一步修改和優化 ### 🔧 技術特色 - **純 HTML/CSS/JavaScript**：無需額外依賴，直接開啟即可使用 - **現代化 UI**：採用漸層背景、卡片式設計、平滑動畫效果 - **繁體中文介面**：完全本地化的使用者介面 - **跨平台相容**：支援 Windows、Mac、Linux 等各種作業系統 ### 📁 檔案結構 ``` Awesome-Nano-Banana-images/ ├── index_zh_tw.html # 主要互動頁面 ├── images/ # 圖片資料夾 │ ├── case1/ # 案例 1 圖片 │ ├── case2/ # 案例 2 圖片 │ └── ... # 其他案例圖片 ├── README.md # 原始說明文件 └── README_en.md # 英文說明文件 ``` ### 🤝 貢獻 如果您發現任何問題或有改進建議，歡迎提出 Issue 或 Pull Request！ ### 📄 授權 本專案採用 CC BY 4.0 授權條款。 --- **享受探索 Nano-Banana 的無限創意可能！** 🎨✨
